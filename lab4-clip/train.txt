Note: If the COCO2014 dataset is in another location, 
please change the BASE_ROOT variable in the dataset.py file.

To run training for the non-augmentation model:
1. Navigate to the non_augmentation directory:
   cd non_augmentation
2. Run the training script:
   python train.py

To run training for the augmentation model:
1. Navigate to the augmentation directory:
   cd augmentation
2. Run the training script:
   python train_augmentation.py
